# Apostila Completa de Claude

Apostila técnica e didática sobre **Claude** (Anthropic) — do modelo de linguagem por baixo
até a arquitetura de agentes, o Claude Code, o MCP e a operação em produção. Escrita para quem
quer entender Claude de verdade, não apenas apertar botões.

Serve também como material de estudo para a certificação **Claude Certified Architect –
Foundations (CCA-F)**.

## Conteúdo

1. **Fundamentos** — o que é um LLM, tokens, context window, a família de modelos Claude, plataformas e pricing.
2. **Prompt Engineering** — estrutura de prompts, técnicas avançadas, raciocínio adaptativo (effort), debugging.
3. **Otimização e Performance** — redução de tokens, prompt caching, memória vs. contexto, custos, multimodalidade.
4. **Integrações e Extensões** — capacidades nativas, Graphify, integrações populares, orquestração e MCP.
5. **Segurança, Legal e Ética** — propriedade intelectual, prompt injection, alucinações, compliance.
6. **Prática e Referência** — casos de uso, boas práticas, troubleshooting, referência rápida.
7. **Integração e Resolução de Problemas** — arquitetura de referência, erros de API, guia de diagnóstico.
8. **Arquitetura de Agentes, Claude Code e MCP em Profundidade** — loop agêntico, subagentes, configuração do Claude Code, saída estruturada, design de ferramentas/MCP, gestão de contexto e a trilha CCA-F.
9. **Claude Code na Prática** — instalação, `settings.json`, custos (`/cost`, `/compact`, `/clear`), hooks, observabilidade, agendamento e integração com editores.

Mais um **Apêndice** com glossário e recursos úteis.
